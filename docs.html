Getting Started

New to FineFoundry? Follow this path to get productive in minutes.

Quick Start

End-to-end walkthrough: install, launch the GUI, and run your first scrape & training job.

  1. Install Python 3.10+ and clone the repo.
  2. Install uv (pip install uv) if needed.
  3. Run uv run src/main.py (handles deps automatically).
  4. Or use a venv + pip: pip install -r requirements.txt then python src/main.py.

Deployment

Run FineFoundry in containers or cloud environments.

Docker & Local Training

Run local training jobs using the Unsloth-based trainer image.

  • Default image: docker.io/sbussiso/unsloth-trainer:latest.
  • Volume mounts for datasets and adapter outputs.
  • Same LoRA stack as RunPod (PyTorch, Transformers, PEFT, bitsandbytes).
Docker deployment docs
RunPod & Cloud Training

Run training jobs on RunPod pods with the Unsloth trainer image.

  • Automated network volume and pod template setup.
  • Same Unsloth-based LoRA stack as local Docker.
  • Outputs written to /data/outputs/... on network volume.

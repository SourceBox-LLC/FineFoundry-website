Quick Start

Installation

Clone and set up the environment

git clone https://github.com/SourceBox-LLC/FineFoundry.git
cd FineFoundry

# Create virtual environment
python -m venv venv

# Activate (Windows PowerShell)
./venv/Scripts/Activate.ps1

# Activate (macOS/Linux)
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
Launch the App

Start the desktop application

# Run the GUI
python src/main.py

# Or use Flet directly
flet run src/main.py

The desktop window will open with tabs for Scrape, Build/Publish, Training, Merge, Analysis, and Settings.

Prerequisites: Python 3.10+ on Windows, macOS, or Linux. Optional: Hugging Face account for Hub integration.

Usage Guide

Scrape Tab

Scrape conversational data from multiple sources

  • Select boards/sources to scrape
  • Configure parameters (max threads, pairs, delay)
  • Monitor live progress and stats
  • Preview dataset before saving
Build/Publish Tab

Create and publish datasets to Hub

  • Build train/val/test splits with sliders
  • Configure shuffle, seed, and filters
  • Push to Hugging Face Hub
  • Auto-generate dataset cards
Training Tab

Fine-tune models on RunPod or locally

  • Select dataset source and base model
  • Configure hyperparameters and LoRA
  • Train on RunPod pods or local Docker
  • Auto-resume and Hub push support
Merge Datasets Tab

Combine multiple datasets seamlessly

  • Merge JSON files and Hub datasets
  • Automatic column mapping
  • Filter empty rows and normalize
  • Download merged results
Dataset Analysis Tab

Comprehensive dataset insights

  • Toggle modules: sentiment, toxicity, etc.
  • Detect duplicates and data leakage
  • Analyze class balance and distribution
  • View readability and alignment metrics
Settings Tab

Configure integrations and preferences

  • Save Hugging Face access token
  • Configure proxy settings per scraper
  • Set up RunPod API key
  • Ollama integration for dataset cards

CLI & Programmatic Usage

FineFoundry also supports command-line and programmatic usage for automation and scripting.

Reddit Scraper CLI
python src/scrapers/reddit_scraper.py \
  --url https://www.reddit.com/r/AskReddit/ \
  --max-posts 50 \
  --mode contextual \
  --pairs-path reddit_pairs.json
Dataset Builder CLI
# Configure constants in src/save_dataset.py
python src/save_dataset.py
Programmatic 4chan Scraper
from scrapers.fourchan_scraper import scrape

pairs = scrape(
    board="pol",
    max_threads=150,
    max_pairs=5000,
    mode="contextual",
    strategy="cumulative"
)
